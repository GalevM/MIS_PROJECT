# 📱 Моја Општина – Мобилна апликација

## 1. Вовед
**Моја Општина** е мобилна апликација наменета за граѓаните и администрацијата на општини во Македонија.  
Овозможува транспарентно пријавување на проблеми во јавниот простор, нивно следење во реално време и ефикасно управување од страна на општинската администрација.

**Платформа:** Flutter (Android/iOS)  
**Јазик:** Dart  
**Backend:** Firebase (Auth, Firestore, Storage)  
**Мапи:** Flutter Map + Geoapify API  
**Медиуми:** Cloudinary  

### Цели
- Лесно пријавување проблеми (дупки, осветлување, отпад итн.)  
- Јавна мапа со статуси  
- Информирање преку известувања и анкети  
- Административно управување со пријави  

---

## 2. Архитектура
Апликацијата користи **клиент-сервер архитектура**:  
- **UI слој:** Flutter widgets, теми, стилови  
- **State слој:** Riverpod провајдери  
- **Data слој:** Firestore CRUD, Firebase Auth, Cloudinary upload  

### Технолошки стек
| Технологија       | Намена                          |
|-------------------|---------------------------------|
| Flutter/Dart      | Frontend – UI и логика          |
| Riverpod          | State Management                |
| GoRouter          | Routing                         |
| Firebase Auth     | Автентикација                   |
| Firestore         | NoSQL база                      |
| Firebase Storage  | Складирање фајлови              |
| Flutter Map       | Интерактивна мапа               |
| Geoapify API      | Геокодирање и autocomplete      |
| Cloudinary        | Складирање и оптимизација слики |

### Шема на податоци (Firestore – reports)
- **id**: String – уникатен идентификатор  
- **category**: String – категорија на проблем  
- **description**: String – опис  
- **imageUrls**: List<String> – до 3 слики (Cloudinary)  
- **latitude/longitude**: double – координати  
- **address**: String – адреса (Geoapify)  
- **status**: String – примено / во тек / решено  
- **createdAt**: Timestamp – датум и време  
- **userId**: String – UID на корисникот  

---

## 3. Главни функционалности

### Автентикација
- Регистрација и најава со е-пошта и лозинка  
- Автоматска детекција на администраторски привилегии  

### Пријавување проблем
- Категорија, опис, до 3 фотографии  
- GPS или рачна адреса со autocomplete  
- Визуелен избор на мапа  

### Јавна мапа
- Маркери со боја според статус:  
  - Жолта → Примено  
  - Сина → Во тек  
  - Зелена → Решено  
- Филтрирање по категорија/статус  
- Детален преглед со опис, слики, адреса  

### Известувања
- Администратор испраќа известувања (работи, настани, итни соопштенија)  

### Анкети
- Креирање анкети  
- Резултати во реално време  

### Административен панел
- Преглед и филтрирање на пријави  
- Промена на статус  
- Статистики  
- Управување со анкети и известувања
